# خطوط العربية للـ PDF

مولّد PDF يعرض العربية بشكل صحيح فقط عند وجود خط عربي هنا.
ضع أحد هذه الملفات في هذا المجلد (public/fonts) ثم أعد بناء المشروع:

- NotoSansArabic-Regular.ttf   (موصى به — تغطية كاملة)
- Amiri-Regular.ttf
- Tajawal-Regular.ttf
- Cairo-Regular.ttf

مصدر مجاني: https://fonts.google.com (نزّل TTF).
بدون الخط، تبقى ملفات PDF تعمل لكن النص العربي قد لا يظهر بشكل مثالي؛
أما Word و Excel و PowerPoint فتعرض العربية بشكل صحيح تلقائياً.
